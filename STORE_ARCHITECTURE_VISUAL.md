## рҹ“Ҡ DATA RELATIONSHIPS

```
USER
  в”ңв”Җ Can have many WISHLISTS
  в”ңв”Җ Can have many CART items
  в”ңв”Җ Can make many ORDERS
  в”ңв”Җ Can write many REVIEWS
  в””в”Җ Can participate in many CHATS

SELLER (User with seller_role=true)
  в”ңв”Җ Can list many BOOKS
  в”ңв”Җ Can receive many ORDERS
  в”ңв”Җ Can participate in many CHATS
  в””в”Җ Can have many REVIEWS

BOOK
  в”ңв”Җ Belongs to one SELLER
  в”ңв”Җ Can have many REVIEWS
  в”ңв”Җ Can be in many WISHLISTS
  в”ңв”Җ Can be in many CARTS
  в”ңв”Җ Can be in many ORDER_ITEMS
  в””в”Җ Can have many IMAGES

ORDER
  в”ңв”Җ Belongs to one BUYER
  в”ңв”Җ Contains many ORDER_ITEMS
  в””в”Җ Each ORDER_ITEM references a BOOK & SELLER

CHAT
  в”ңв”Җ Between one BUYER and one SELLER
  в”ңв”Җ About one BOOK (optional)
  в””в”Җ Contains many MESSAGES
```

# 📄 Quiz Questions PDF Format Guide

## HOW TO FORMAT YOUR QUESTIONS PDF

When creating a PDF for question upload, use this exact format:

```
1. What is the purpose of the SUM function in Excel?

A) To multiply numbers
B) To add numbers together
C) To divide numbers
D) To find the average

Answer: B

2. Which shortcut saves a file in Excel?

A) Ctrl+P
B) Ctrl+Z
C) Ctrl+S
D) Ctrl+C

Answer: C
```

## RULES

| Rule | Details |
|------|---------|
| **Question format** | Start with a number: `1.` or `1)` or `Q1.` |
| **Options format** | Use `A)`, `A.`, or `A ` before each option |
| **Answer format** | Write `Answer: B` or `Ans: C` or `Correct: A` |
| **4 options required** | Each question should have A, B, C, D |
| **Plain text PDF** | Made from Word/Google Docs, not a scanned image |

## TIPS
- ✅ Up to 200+ questions can be parsed at once  
- ✅ Use "Replace existing questions" to clear old quiz and add new ones
- ✅ Use "Append" (uncheck replace) to add to existing questions
- ❌ Do NOT use scanned images — text must be machine-readable
